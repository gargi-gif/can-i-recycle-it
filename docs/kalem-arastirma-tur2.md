## Lider kararları (FLAG çözümleri)
1. Kâğıt havlu → false. sifiratik.gov.tr otomatik özeti uluslararası konsensüsle çelişti; bilimsel
   gerekçe (kısa lif + kir) baskın. Kesin TR hükmü iddia edilmeyecek.
2. Ahşap askı kapsam dışı (kaynak yok); kalem plastik/tel askı.
3. Yeni kategori: **other / "General Waste"** (bebek bezi, seramik, maske). Renk: TR standardındaki
   "diğer = siyah/koyu gri" (dosab kaynağı, arastirma-bulgulari.md) → stone ailesi, metal(zinc)'den ayrışır.
